# 🧮 Simple Standard & Scientific Calculator

Sebuah aplikasi web kalkulator sederhana yang menggabungkan fungsi kalkulator **Standar** dan **Ilmiah (Scientific)** dalam satu antarmuka yang bersih dan responsif. Proyek ini dibangun murni menggunakan teknologi web dasar: **HTML5, CSS3, dan Vanilla JavaScript**.

---

## 🚀 Fitur Utama

* **Dual Mode:** Beralih dengan mudah antara mode kalkulator Standar dan Ilmiah.
* **Operasi Standar:** Penjumlahan, pengurangan, perkalian, pembagian, persen, dan hapus karakter (`backspace`).
* **Operasi Ilmiah:** Mendukung fungsi trigonometri (sin, cos, tan), logaritma (log, ln), akar kuadrat ($\sqrt{x}$), pangkat ($x^y$), nilai mutlak, eksponensial, serta konstanta matematika ($\pi$ dan $e$).
* **Desain Responsif:** Tampilan yang nyaman dilihat dan digunakan baik di perangkat *mobile* (smartphone) maupun *desktop*.
* **Keyboard Support:** Bisa dioperasikan menggunakan *numpad* atau keyboard fisik (opsional, jika kamu mengimplementasikannya).

---

## 🛠️ Teknologi yang Digunakan

* **HTML5:** Untuk struktur semantik dari tombol dan layar kalkulator.
* **CSS3:** Untuk styling, tata letak menggunakan Grid/Flexbox, dan efek transisi yang halus.
* **JavaScript (ES6+):** Untuk logika perhitungan matematika dan manipulasi DOM secara *real-time*.
